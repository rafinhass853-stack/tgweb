# TGWeb - Sistema de Gestão de Frota

Sistema web para gerenciamento de motoristas, veículos, programações e integração com WhatsApp.

## Tecnologias

- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express + Socket.io
- **Banco**: Firebase (Auth + Firestore)
- **WhatsApp**: whatsapp-web.js
- **Mapas**: Leaflet + React Leaflet

## Como rodar

```bash
# Instalar dependências
npm install

# Rodar backend + frontend juntos
npm run dev:all
